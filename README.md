# Currency Bot

Бот для перевода одной валюты в другую. Отслеживает курсы валют и обновляет их каждый день.

Протестировать бота можно [тут](https://t.me/techjob_task_bot).

## Технологии
- aiohttp
- aiogram
- apscheduler
- Docker
- Redis

## Фичи
- [x] Получение и сохранение курсов валют в Redis
- [x] /exchange
